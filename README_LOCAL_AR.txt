تواصل برو - نسخة التسليم المحلية v6.45.2
=====================================

هذه النسخة تم تشديدها أمنيًا وتحسين تهيئتها للتشغيل المحلي وLAN بشكل أوضح وأكثر أمانًا.

أهم التحسينات:
- إصلاح صلاحيات Socket الحرجة على الرسائل والتفاعلات والقراءة والحذف والتثبيت
- إزالة الأسرار الفعلية والشهادات الخاصة من الحزمة
- تعطيل الوصول العام للملفات المرفوعة افتراضيًا وربطه بمسار محمي داخل API
- تحسين إعدادات LAN والاعتماد على hostname/runtime بدل localhost الصلب داخل الواجهة
- تشديد مسارات المكالمات المباشرة لمنع الإزعاج من مستخدمين بلا محادثة مشتركة
- تحسين Electron ليفضّل localhost ويحمّل build الواجهة عند توفره

قبل أول تشغيل:
1) افتح المجلد backend
2) انسخ الملف `.env.example` إلى `.env`
3) ضع قيمة قوية لـ JWT_SECRET
4) إذا أردت HTTPS، أنشئ شهاداتك الخاصة داخل certs وعدّل المسارات

أوامر التشغيل:
- setup.bat         تهيئة أول مرة
- start.bat         تشغيل الواجهة + الخادم
- start-all.bat     تشغيل الواجهة + الخادم + Electron + الجوال
- start-desktop.bat تشغيل Electron بعد تشغيل الواجهة والخادم
- start-mobile.bat  تشغيل Expo للجوال
- stop.bat          إيقاف نوافذ التشغيل المعروفة

ملاحظات مهمة:
- الواجهة تعمل افتراضيًا على http://localhost:3020
- الـ API يعمل افتراضيًا على http://localhost:4000
- للوصول من LAN استخدم IP الجهاز الحالي بدل localhost
- الرفع العام للملفات معطّل افتراضيًا لأسباب أمنية
- بعد أي تعديل على Prisma استخدم:
  npx prisma generate
  npx prisma db push
