قائمة التحقق النهائية للنشر
===========================

قبل الرفع إلى GitHub:
[ ] تأكد أنك تعمل من هذه الحزمة النظيفة
[ ] لا تضف ملفات .env الحقيقية
[ ] لا تضف شهادات محلية أو مفاتيح

بعد رفع GitHub:
[ ] أنشئ مشروع Railway جديد
[ ] أضف PostgreSQL
[ ] اضبط خدمة backend على /backend
[ ] ضع Variables الخاصة بـ Railway
[ ] اختبر /api/health

بعد ذلك:
[ ] أنشئ مشروع Vercel جديد
[ ] اضبط Root Directory = frontend
[ ] ضع Environment Variables الخاصة بـ Vercel
[ ] أعد النشر إذا عدلت المتغيرات

بعد النشر:
[ ] افتح موقع Vercel
[ ] جرّب التسجيل وتسجيل الدخول
[ ] تأكد أن الطلبات لا تذهب إلى /api/api
[ ] إذا ظهر CORS عدّل ALLOWED_ORIGINS في Railway
