تواصل برو 6.1 - تعديلات جذرية

1) إضافة publicId مكون من 12 رقم في نموذج المستخدم ويظهر في الواجهة.
2) البحث عن المستخدمين صار يدعم الاسم + الإيميل + publicId.
3) إنشاء جلسة Session عند التسجيل وتسجيل الدخول والتحقق منها في كل طلب وSocket.
4) إزالة الاعتماد على JWT secret افتراضي وإلزام ضبطه من backend/.env.
5) تشديد CORS عبر ALLOWED_ORIGINS بدل السماح للجميع.
6) حماية Audit Log من الوصول خارج أعضاء المحادثة.
7) حماية todos / calendar / reminders / stars بناءً على عضوية المحادثة.
8) حماية webhooks والمنتجات بالأدوار المناسبة داخل المحادثة.
9) حماية broadcast delete حسب المالك.
10) حماية admin stats عبر ADMIN_EMAILS.
11) إزالة IP الثابت من frontend / electron / mobile / start scripts / invite URLs.
12) تشديد رفع الملفات عبر MIME whitelist للأفاتار والملفات العامة.

مهم بعد فك الضغط:
- عدل JWT_SECRET في backend/.env قبل التشغيل.
- نفذ prisma generate و prisma db push بعد تثبيت الحزم.

- Draft messages are saved per conversation and restored automatically.
- Auto-resizing message textarea added.
- Mobile back button enabled in chat header.
- Connection banner added for offline / reconnecting states.
- Sticky composer and safe-area mobile spacing improved.
- Lazy loading enabled for chat images.
