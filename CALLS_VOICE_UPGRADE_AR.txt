تم في هذه النسخة تحسين ما يلي:

1) المكالمات:
- إضافة حالات أوضح للمكالمة: incoming / calling / connecting / connected / ended / busy / rejected / missed.
- إضافة مؤقت مرئي للمكالمة.
- إضافة أزرار كتم الميكروفون وتشغيل/إيقاف الكاميرا أثناء المكالمة.
- تحسين شاشة المكالمة بصريًا مع بطاقات حالة أوضح.
- إضافة timeout للمكالمات غير المجابة.
- تحسين تحديث سجل المكالمات بعد انتهاء المكالمة.
- إضافة تحديثات حالة المكالمة عبر Socket.IO.

2) التسجيل الصوتي:
- لم يعد التسجيل الصوتي يُرسل فورًا دون مراجعة.
- تمت إضافة حالات: تسجيل / إيقاف مؤقت / استئناف / إنهاء / معاينة / إرسال / حذف.
- إضافة مؤقت أثناء التسجيل.
- إضافة waveform مبسط أثناء التسجيل والحفظ داخل meta للرسالة الصوتية.
- إضافة معاينة للصوت قبل الإرسال.
- تحسين عرض الرسالة الصوتية داخل المحادثة بمشغل مخصص أوضح.

3) الرسائل:
- حفظ meta للرسائل الصوتية وإرجاعها من الخادم.
- عرض مدة الرسالة الصوتية داخل واجهة الدردشة.

ملاحظات:
- هذه التحسينات موجهة للويب داخل البنية الحالية.
- المكالمات ما زالت تعتمد على WebRTC مباشر وليست بنية TURN/SFU إنتاجية كاملة.
- في بيئات الشبكات الصعبة قد تحتاج إضافة TURN server لاحقًا.
