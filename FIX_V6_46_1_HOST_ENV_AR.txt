إصلاح v6.46.1

تم إصلاح مشكلة تشغيل الواجهة على Windows التي كانت تظهر كالتالي:
Error: getaddrinfo ENOTFOUND 0.0.0.0

السبب:
- استخدام أوامر BAT بصيغة set VAR=value && npm start
- هذا النمط على cmd.exe قد يضيف فراغًا زائدًا داخل قيمة المتغير
- فتصل قيمة HOST كأنها 0.0.0.0 مع مسافة في النهاية

ما تم إصلاحه:
- تحويل جميع أوامر set داخل ملفات BAT إلى الصيغة الآمنة: set "VAR=value"
- اعتماد HOST=11.0.0.103 للواجهة بدل 0.0.0.0
- تحديث frontend/.env ليستخدم 11.0.0.103 افتراضيًا
- توحيد الإصدار إلى 6.46.1
