تواصل برو - نسخة Mobile Native متقدمة
=====================================

هذه النسخة أصبحت تدعم بشكل أفضل:
- واجهات Native للمحادثات والملف الشخصي
- مكالمات فردية وجماعية Native/WebRTC بشكل أقوى
- إشعارات مكالمات خلفية عبر Expo Notifications
- القبول / الرفض من الإشعار
- استعادة الجلسة بعد الرجوع من الخلفية
- بناء Release APK محسن بدل الاقتصار على Debug

التشغيل:
1) افتح المجلد mobile-app
2) نفذ npm install
3) للتطوير: npm start
4) للبناء السحابي APK: build-apk-eas.bat
5) للبناء المحلي Release APK: build-apk-local.bat أو build-apk.bat

ملاحظات:
- يجب أن يكون backend يعمل على عنوان API الصحيح مثل:
  https://localhost:4000
- إشعارات الخلفية الكاملة تحتاج جهازًا حقيقيًا.
- لأفضل إشعارات إنتاج، أكمل تهيئة Expo Push / FCM عند البناء النهائي عبر EAS.
- تم إضافة سجل Push Tokens في قاعدة SQLite الداخلية (push_tokens) مع ترحيل تلقائي من JSON القديم عند الحاجة بشكل خفيف بدون تعديل قاعدة البيانات.
