ملف ملخص التسليم النهائي - Twasol Pro v6.45.0
===========================================

النسخة الحالية تشمل:
- نظام مستخدمين مع publicId
- جلسات وأجهزة آمنة نسبيًا بدون كشف التوكنات
- تحسينات الدردشة والمرفقات والمعرض
- تحسينات المجموعات والقنوات والدعوات
- مركز إشراف وأدوات إدارة داخل الدردشة
- مكالمات فردية وجماعية مع تشديد التحقق في signaling
- إشعارات خلفية واستمرارية أفضل على الجوال
- تخزين آمن للتوكن في الجوال عبر SecureStore
- تشفير PIN وعدم إرجاعه من الـ API
- Webhooks محمية مع إخفاء الأسرار
- نقل Push Tokens إلى SQLite داخلي بدل JSON
- تحسينات بناء APK وفرض HTTPS افتراضيًا
- جولة تحصين أمني وتنظيف بنيوي وتسليم

- setup.bat
- start.bat
- start-all.bat
- start-https.bat
- stop.bat
