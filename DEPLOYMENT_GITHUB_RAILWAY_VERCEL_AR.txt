نشر المشروع: GitHub + Railway + Vercel

1) GitHub
- ارفع المشروع كاملًا إلى المستودع العام: proometoo-hub/twasol

2) Railway
- أنشئ Project جديد
- أضف PostgreSQL service
- أضف Service من GitHub repo نفسه
- Root Directory = backend
- Build Command = npm run build
- Start Command = npm run railway:start
- أضف Volume للمسار /app/uploads
- أضف متغيرات البيئة من backend/.env.example

3) Vercel
- Import Git Repository
- Root Directory = frontend
- Framework = Create React App
- Build Command = npm run build
- Output Directory = build
- Environment Variables:
  REACT_APP_API_URL=https://your-backend.up.railway.app
  REACT_APP_SOCKET_URL=https://your-backend.up.railway.app

4) بعد أول نشر
- خذ رابط Vercel وضعه داخل ALLOWED_ORIGINS في Railway
- أعد نشر Railway ثم Vercel
