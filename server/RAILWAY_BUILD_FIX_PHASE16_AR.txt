تم إصلاح مشكلة فشل البناء على Railway في هذه النسخة.

ما تم تعديله:
- إضافة script باسم build داخل server/package.json حتى لا يفشل الأمر npm run build.
- جعل build في هذه النسخة خطوة no-op لأن الخادم الحالي لا يحتاج TypeScript أو Prisma generate أثناء البناء.
- تحديث server/nixpacks.toml و server/railway.json ليبقيا متوافقين مع Railway.

ملاحظة مهمة:
هذه النسخة ما زالت تستخدم SQLite في التشغيل الحالي للخادم.
DATABASE_URL أو سكربتات PostgreSQL لا تستخدم في runtime لهذه النسخة.
